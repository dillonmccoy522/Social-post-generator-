Developer Handoff Document — Built 2026-07-05

What This Is

Social Media Content Generator

A web app for generating 3 social media posts per week for agency clients using Claude AI. Manages clients, generates Instagram/Facebook captions with hashtags and CTAs, and stores post history.

Current state: Fully functional local app. Built to be migrated to a hosted environment (Railway).

Environment Variables

ANTHROPIC_API_KEY REQUIRED
Anthropic API key for Claude post generation. Get from console.anthropic.com → API Keys. Used in routes/generate.js.
For Railway deployment: Add ANTHROPIC_API_KEY as an environment variable in the Railway project settings. Do not commit .env to git — it is gitignored.

Railway Deployment Notes

  • Set ANTHROPIC_API_KEY in Railway environment variables
  • SQLite works on Railway but the database resets on each deploy — migrate to PostgreSQL for persistence
  • No build step needed — npm start runs the app directly
  • Port: app reads process.env.PORT with fallback to 3000 — Railway sets this automatically
  • The data/ directory is auto-created on first run
